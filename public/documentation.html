{ "ID": 1, "Title": 'Movie Title', "Actors": ['Actor 1', 'Actor 2'], ... }
Get Genre by Name /genre/:genreName Return the descripion of a genre by its title GET :genreName as Name of the Genre N/A JSON Object:
{ "ID":1, "Title": "Genre Title", "Description": "Genre Description" }

{ "ID": 1, "Name": "Director Name", "Bio": "Director Bio", "Birth": 01/01/1970 }
