API Endpoints

Operation Endpoint URL Function HTTP Method Parameters Request Data Format and Example Return Data Format and Example
Get Movies /movies Return all movies in the database GET N/A N/A JSON Object:
[ ... { "ID": 1, "Title": 'Movie Title', "Actors": ['Actor 1', 'Actor 2'], ...} ... ]
Get Movie by Title /movies/:title Return details of a single movie by its title GET :title as Movie Title N/A JSON Object:
{ "ID": 1, "Title": 'Movie Title', "Actors": ['Actor 1', 'Actor 2'], ... }
Get Genre by Name /genre/:genreName Return the descripion of a genre by its title GET :genreName as Name of the Genre N/A JSON Object:
{ "ID":1, "Title": "Genre Title", "Description": "Genre Description" }
Get Director by Name /directors/:directorName Return information on a specific director by name GET :directorName as Name of the Director N/A JSON Object:
{ "ID": 1, "Name": "Director Name", "Bio": "Director Bio", "Birth": 01/01/1970 }
Get Users /users Return all users in the database GET N/A N/A JSON Object:
{ "ID": 1, "Username": "Steven", "Favorites": "["Movie 1", "Movie 2"...]" }
Get Users /users/:Username Return a single user in the database by it's Username GET :Username as Name of User N/A JSON Object:
{ "ID": 1, "Username": "Steven", "Favorites": "["Movie 1", "Movie 2"...]" }
Create User /users Create a User POST N/A JSON Object:
{ "Username": "Steven", "Favorites": "["Movie 1", "Movie 2"...]" } 		JSON Object:
{ "ID": 1, "Username": "Steven", "Favorites": "["Movie 1", "Movie 2"...]" }
Update User /users/:Username Update user details PUT :Username as Name of User JSON Object:
{ "Username": "Steven", "Favorites": "["Movie 3", "Movie 4"...]" }} 		JSON Object of updated User:
{ "ID": 1, "Username": "Steven", "Favorites": "["Movie 3", "Movie 4"...]" }
Add Movie to Favorites /users/:id/movies/:movieID Add movie to favorites POST :Username as Name of User
:movieID as the ObjectID of the movie 		N/A JSON Object of updated User:
{ "ID": 1, "Username": "Steven", "Favorites": "["Movie 3", "Movie 4"...]" }
Remove Movie from Favorites /users/:id/movies/:movieID Remove movie from favorites DELETE :id as the ID of the user, :movieTitle as the title to be added to favorites N/A JSON Object of updated User:
{ "ID": 1, "Username": "Steven", "Favorites": "["Movie 3", "Movie 4"...]" }
Delete User /users/:id Remove user DELETE :id as the specific ID of the user N/A Text Reponse with Status Message:
Success - User has been removed
Failure - User couldnt be found
